# MediaDB

MediaDB (terrible name) is a native Android app built with the TMDB API for retrieving popular movies and TV shows, as well as searching them and viewing details.

- Kotlin
- MVP
- Dagger2
- RxJava
- Retrofit
- Room
- Glide
- Timber
- Lottie
